# Logos de Fundadores

Guardá acá el logo de cada barbería del Programa Fundadores, con el nombre
del slug: `leo-cuts.png`, `otra-barberia.png`, etc.

Después, en `src/data/founders.ts`, poné la ruta en `logoSrc`:

    logoSrc: "/fundadores/leo-cuts.png",

Si `logoSrc` queda en `null` se muestran las iniciales de la barbería.
Formato recomendado: PNG o WebP cuadrado, mínimo 256×256.
